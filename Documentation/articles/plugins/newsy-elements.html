Newsy Elements

Newsy uses content blocks for making pages. This is a very flexible method for creating awesome pages and provides a wide customizability. Use WPBakery Page Builder or Elementor Page Builder plugin to manage these content blocks in your pages. Most of Newsy blocks are exclusively designed for Newsy theme.

Newsy provides you 50+ exclusive elements for free (48 content blocks and 7 widgets). Newsy elements cover large range of styles and functionalities and also are fully customizable.

Here is the list of Newsy content blocks:

  • Blocks (14 item)
  • Lists (18 item)
  • Grids (13 item)
  • Slider (3 item)
  • Widget Blocks (Video Playlist, social counter, newsletter, instagram) (7 item)

If you want to see demos of the Newsy content blocks, go to https://demo.akbilisim.com/newsy/ and in main menu go to Menu → Elements, look at below image.

Install & Activate

To enable content blocks on your website, you just need to install Newsy Elements plugin. You can easily install and activate this plugin by following Install Required Plugins.

Setting

Right after you’ve done with the plugin activation, next you can doing setup global block and module options to fit with your website’s need. You can access these options by simply go to Newsy → Theme Options → Blocks & Modules.

You also can use Newsy Elements with your preferred page builder and widget areas.
