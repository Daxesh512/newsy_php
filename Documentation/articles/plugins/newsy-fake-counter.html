Newsy Fake Counter

Fake counters functionality allows you to display a fake count instead of the real score, so you don’t have to start from 0. This can be set only globally.

Install & Activate

To enable fake counter feature on your website, you just need to install Newsy – Fake Counter plugin. You can easily install and activate this plugin from Plugin Dashboard by go to Newsy Dashboard → Install Plugins.

Setting

Right after you’ve done with the plugin activation, next you can setup the plugin settings to fit with your website’s need. In this part we will guide you how to setup fake counter with Newsy.

You can access fake counter options by simply go to Newsy Dashboard → Fake Counter Options

  1. Fake View Counts
    - Initial View Count
    This value will increase your initial view counter. This value will affect social share, post voting, reaction voting values.
  2. Fake Share Counts
    -Social Share Value base on View
    We increase your social share value base on percentage of your view. You can set fake percentage for each provider in the same place.
  3. Fake Reaction Counts
    - Reaction Value base on View
    We increase your reaction voting value base on percentage of your view. You can set fake percentage for each reaction in the same place.
  4. Fake Voting Counts
    - Up/Like Voting Count Base
    Base value for up voting count.
    - Up/Like Voting Value base on View
    We increase your post voting value base on percentage of post view.
    - Down/Dislike Voting Count Base
    Base value for down voting count.
    - Down/Dislike Voting Value base on View
    We increase your post voting value base on percentage of post view.
