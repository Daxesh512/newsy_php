BuzzEditor

The BuzzEditor give users the ability to create new posts from the site frontend so that the user does not need to enter the backend admin panel to do action.

In this part, we will briefly guide you how to use Frontend Submit feature with BuzzEditor Plugin easily.

Install & Activate

To enable the Frontend Submit feature on your website, you just need to install  BuzzEditor plugin. You can easily install and activate this plugin from Plugin Dashboard by go to Newsy Dashboard → Install Plugins

Setting

Right after you’ve done with the plugin activation, next you can doing setup frontend editor options to fit with your website’s need. You can access social counter options by simply go to Newsy Dashboard → BuzzEditor Options.

General Tab

  1. Enable Frontend Editor
    On this option, you can enable frontend editor option.
  2. Frontend Endpoint
    On this option, you can change frontend editor endpoint.
  3. Enable Frontend Editor for User Roles
    Choose Frontend editor access by user roles. You may want to allow access only for Editor user role.
  4. Frontend Editor Layout Style
    Select whether you want a boxed or a full width layout for frontend editor page.

User Permissions Tab

  1. User Can Add and Edit Draft/Pending Posts
    Enable this option to give user permission for adding posts.
  2. User Can Delete Draft/Pending Posts
    Enable this option to give user permission for deleting posts.
  3. User Can Publish Posts
    Enable this option to give user permission for publishing posts without approval.
  4. User Can Edit Published Posts
    Enable this option to give user permission for editing published posts.
  5. User Can Delete Published Posts
    Enable this option to give user permission for deleting published posts.

User Blocks Tab

  1. Enable User Blocks
    Here you can select frontend blocks. User will be able to add only selected blocks. Don't not select any to allow all blocks. Core blocks may depends on another core block, please make sure that you have selected both. Ex: core/column block works with core/columns block.

Advanced Tab

  1. Editor Autosave Interval
    Set autosave time in second for frontend editor.
  2. User Max Upload File Size
    Set default user max file upload size.

Usage

Create a New Post in Frontend

Right after you’ve done with the plugin activation, next you can create a new post in the frontend. You can follow the instructions below :

  1. Go to https://yourdomain.com/editor/.
  2. After that, you will see this Editor Page. In this page, you can make a new post by clicking the Publish button after filling the form.
  3. After submitting a new post, the post will reviewed by your website administrator.
