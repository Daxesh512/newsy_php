Newsy Reaction

Reactions are a simple way for users to express their reactions and feelings to a post.

Install & Activate

To enable reactions on your website, you will need to install both of Newsy Reaction plugin. You can easily install and activate this plugin from Plugin Dashboard by going to Newsy Dashboard → Install Plugins.

Setting

Right after you’ve done with the plugin activation, next you can setup the plugin settings to fit with your website’s need. In this part we will guide you how to setup reactions with Newsy.

You can access reaction options by simply  go to Newsy Dashboard → Reaction Options.

  1. Select Reactions
    You can select specific reaction badges and you can order them. By default all reactions will show up if you don't use this option.
  2. Reaction Badge Threshold
    Assign reaction badge to post if vote count reaches this threshold.
  3. Guest voting is enabled?
    On this option, you can choose on or off the guest voting on the reaction box.

  4. Reaction Box Style?
    On this option, you can select style of reaction icons on the single page.
  5. Reaction Box Accent Color
    On this option, you can select color of reaction icons on the single page.
