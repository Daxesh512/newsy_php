Newsy View Counter

View counter is useful feature that a good website should own for statistical purpose. Newsy has provided a built-in view counter plugin that we branch from most popular view counter plugins. 

Count post views and display them in professional way according to number of views with various options, also compatible with widgets and Visual Composer for displaying popular posts.

Install & Activate

To enable view counter feature on your website, you just need to install Newsy View Counter plugin. You can easily install and activate this plugin from Plugin Dashboard by go to Newsy Dashboard → Install Plugins.

Setting

Right after you’ve done with the plugin activation, next you can setup the plugin settings to fit with your website’s need. In this part we will guide you how to setup view counter with Newsy.

You can access view options by simply go to Newsy Dashboard → Theme Options → Post Page → Post: View Counter Section.

  1. Enable 7 days Views?
    On this option, you can enable weekly most liked posts option.
  2. Custom View Count Icons
    On this option, you can select custom icons for view counts on the post meta.
