Newsy Social Counter

When you install and activate Newsy Social Counter plugin on your website, one new widget, Social Counter, will be added to your widgets. But what this widget does? It shows number of followers (likes, subscribers or whatever it’s called) of your social networks.

Install & Activate

To enable social counter block on your website, you just need to install Newsy Social Counter plugin. You can easily install and activate this plugin from Plugin Dashboard by go to Newsy Dashboard → Install Plugins.

Setting

Right after you’ve done with the plugin activation, next you can doing setup social counter options to fit with your website’s need. You can access social counter options by simply go to Newsy Dashboard → Social Counter Options.

  • Facebook Counter
    - Facebook App ID
    Insert your Facebook app ID. Get the Facebook app ID on this page and create the new app.
    - Facebook App Secret
    Insert your Facebook app secret. Get the Facebook app secret on this page after creating a new app.
    - Facebook Access Token
    Insert your Facebook access token. Get the Facebook access token on this page then go to Graph API Explorer and generate the token
  • Twitter Counter
    - Twitter ID
    Insert default Twitter account as share reference for Twitter sharing a.k.a “… via @username”.
    - Twitter Access Token
    Insert your Twitter access token. Get the Twitter access token on twitter app page.

  • Youtube Counter
    - Youtube ID
    Insert default User: www.youtube.com/user/ENVATO, Channel: www.youtube.com/ channel/UCJr72fY4cTaNZv7WPbvjaSw
    - Youtube Key
    Insert your Youtube Key. You can register Google API Key here for YouTube.
  • Instagram Counter
    - Instagram ID
    Insert default Instagram account ID.
    - Instagram Access Token
    Insert your Instagram Access Token. Get your Instagram Access Token by clicking link in the description.

  • SoundCloud Counter
    - SoundCloud ID
    Insert default SoundCloud account ID.

  • Pinterest Counter
    - Pinterest ID
    Insert default Pinterest account ID.

  • Steam Counter
    - Steam Group ID
    Insert default Steam group ID.

  • Flickr Counter
    - Flickr ID
    Insert default Flickr account ID.
    - Flickr Access Key
    Insert your Flickr Access Key. 

  • Foursquare Counter
    - Foursquare ID
    Insert default Foursquare account ID.
    - Foursquare Access Key
    Insert your Foursquare Access Key. 

  •  VK Counter
    - VK ID
    Insert default VK account ID.
    - VK Access Key
    Insert your VK Access Key. 

  • Twitch Counter
    - Twitch ID
    Insert default Twitch account ID.
    - Twitch Channel ID
    Insert default Twitch channel ID.
    - Twitch Access Key
    Insert your Twitch Access Key. 

 

Default Counts

Here you can manually set default counts for providers. This options allows you to add your own counts if you don't want to use api keys.
