bbPress

bbPress is a powerful plugin for building forums in WordPress websites. It doesn’t take up many server  resources, adds forums to WP website very easily and also has a built-in feature for preventing spams. It is an extensive plugin and can create custom forums very easily and also keeps flexibility of WordPress itself.

Newsy supports this plugin and you can use it for creating your forums very easily. Let’s take a look at what this plugin gives to us.

Install & Activate

To use bbPress plugin, you can easily install and activate this plugin from Plugin Dashboard by going to Newsy Dashboard → Install Plugins.

Setting 

Right after you’ve done with the plugin activation, next you can setup the plugin settings to fit with your website’s need. You can access forum page options by simply go to Newsy Dashboard → Theme Options → Other Pages → bbPress Page.

You can select layout of forum pages. 

  1. Layout
  2. Primary Sidebar
  3. Layout Style

Usage 

Forums

You can create multiple forums and use them for different topics in different parts of your website. It’s possible to create a child forum for another forum (parent forum). You can see a simple demo of bbPress forums in Newsy here.

Category

Create one category page in bbPress and add as many as forums to it.

Topics

Every forum should contains one or some topics. Users can create new topics in forums and start a discussion.

Replies

Users can reply to topics created by you or other tickets. In this way they discuss each other about the topic.
