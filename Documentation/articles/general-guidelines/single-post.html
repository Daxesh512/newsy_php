Single Post

There’re a few things that we’ll explain to you on this section :

  1. Post Template
  2. Post Format
  3. Post Editor
  4. Primary Category
  5. Single Post Setting
  6. Sponsored Post

1. Post Template

With Newsy, we provide 12 kinds of post template that you can use for your post. Please note, several template may not work for certain post format.
 

  • Standard Format : This format will work best on all template.
  • Video Format : This format will works best with Template 1, Template 2, Template 3, Template 4, Template 7, Template 8 and Template 12.
  • Gallery Format : This format will work best on all template except Template 12.

2. Post Format

With Newsy, we provide 3 kinds of post format that you can use for your post. There are Standard Format, Video Format, and Gallery Format.

As seen on image above, you can choose standard or video post format. Please check the following explanation below about post format :

  1. Standard Format
    This is the default option for post format. With this format, you can also add featured image for your post.
  2. Video Format
    You can also use video for featured post by using video post format. Nothing complicated is required, just paste your video url and it’s done. Please "Post Featured Video" option in metabox.
  3. Gallery Format
    If you want to show image gallery format icon on a post, this post format will fit in your need. With this format, you can use gallery block in post editor.

3. Post Setting

In some cases, you might want to have individual setting for certain post. To achieve that, we provide Single Post Setting that you can find on each post. With this feature you will be able to override Global Single Post Setting. So each post on your site can have its own personal setting.

Note:
If you are not familiar with Global Single Post Setting yet, you can follow this link 

Please check the following explanation about Newsy Post Options below :

    Post Tab

  1. Post Template
    Choose post header template that you want to use for the current post. Please note, several template may not work for certain post format.
  2. Post Thumbnail Size
    Turn on this option to override default image thumbnail size. Please note, if you’re using post template with image full size, this option will be ignored. 
  3. Post Featured Video
    Turn on this option to show featured video for the current post. Please note, if you’re using post video you should video post format. 
    - From the URL:
    Set Video Url,  Video Length options.
    - Upload:
     Set Mp4 Video, Webm Video, Is Gif Video? options.
  4. Post Primary Category
    On certain post, you might use more than one categories. In this case, only the first post category will be used by default. However, if you personally want to use a certain category instead of the one selected by the system, you can setup Primary Category manually for your post.
  5. Show Breadcrumb
    Turn on this option to show breadcrumb for the current post.
  6. Show Featured Image/Video
    Turn on this option to show featured image or video for the current post.
  7. Show Excerpt
    Turn on this option to show except for the current post.
  8. Show Tags
    Turn on this option to show post tag on the below article for the current post.
  9. Show Author Box
    Turn on this option to show author box on the below article for the current post.
  10. Related Posts Type
    Turn on this option to show related post on the below article for the current post.

    Others Tab

  1. Layout
    Choose layout for the current post page.
  2. Primary Sidebar
    Choose page primary sidebar for the current page. Please note, this option will appear and work if you choose Right Sidebar or Left Sidebar on Layout option above.
  3. Secondary Sidebar
    Choose page secondary sidebar for the current post page. Please note, this option will appear and work if you choose Secondary Right Sidebar or Secondary Left Sidebar on Layout option above.
  4. Layout Style
    Select whether you want a boxed or a full width layout for the current post page. Default option shows what default style selected in theme options.
  5. Main Navigation Menu
    Choose custom main navigation menu for the current post page.
