Sidebar & Widgets

There are few things that we will explain to you on this part :

  1. Widget Area 
  2. Add / Remove Widget Area (Sidebar)
  3. Newsy Widgets

1. Widget Area

Newsy comes with 5 pre-defined widget areas that you can use to decorate your website. Here’s the list of pre-defined widget areas :

  1. Primary Sidebar
    This is widget area for primary sidebar that will be visible on Blog & Page that enable sidebar.
  2. Secondary Sidebar
    This is widget area for secondary sidebar that will be visible on Blog & Page that enable sidebar.
  3. Footer - Column 1
    This is widget area for footer at the first column. Please note, this widget area will work if you’re using them as footer element on Footer Builder.
  4. Footer - Column 2
    This is widget area for footer at the second column. Please note, this widget area will work if you’re using them as footer element on Footer Builder.
  5. Footer - Column 3
    This is widget area for footer at the third column. Please note, this widget area will work if you’re using them as footer element on Footer Builder.

2. Add / Remove Widget Area

We also provide an option to add and remove the widget area. To create new widget area, you just need to;

  1. Go to WordPress Admin → Appereance → Widgets then you will notice Add or Remove Widget Area button in there.
  2. As you can see on the image above, you can create new widget area by clicking Create Widget Area button or remove certain widget area by clicking red icon.
  3. Don’t forget to hit Save Widget button after you’ve done.
  4. As seen on the image above, this panel will show up when you create a new widget. You just need to insert the name of the new widget and don’t forget to hit Add Widget button to finish the process.

  10. Newsy List 2

  12. Newsy List 3
