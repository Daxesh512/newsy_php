Installing Theme

Before all else, you must setup a WordPress version on your server. (Read more at How to Install Wordpress.)

After downloading the package, you will have “newsy” folder and “newsy.zip” file inside the Theme package. (Read more at What’s included.) You can either choose to upload the Theme to your server via ftp (just upload the folder “newsy” into the folder wp-content/themes) or via WordPress upload function (Appearance > Themes > Install Themes > Upload) using the “newsy.zip” file.

Install via WP upload function

  • Download “All Files and Documentation” package from Themeforest download page and unzip it.
  • In unzipped folder, you will find newsy.zip file. Keep it as it is, do not unzip it again.
  • In your WordPress Dashboard go to Appearance → Themes.
  • Click on Add New Theme.
  • Click Upload Theme and browse the newsy.zip file and click Install Now.
  • After installation is complete click on Activate Theme. 
  • Right after installing the theme, you will be prompted to install the required plugins. You can follow the instruction right here.
Important:
One common mistake in installing theme via WordPress is, trying to upload downloaded package from ThemeForest.net as theme file. This leads to an error from WordPress; this error usually seems like this “The package could not be installed. The theme is missing the style.css stylesheet.”. Just upload newsy.zip file, not the whole downloaded package. Read more at What’s included.

Install via FTP

  • Download ZIP package from Themeforest and unzip it.
  • In unzipped folder, you will find newsy.zip file. Please unzip it again.
  • Upload the unzipped "newsy" folder in wp-content/themes directory.
  • After this go to Appearance/Themes and activate the theme. 
  • Right after installing the theme, you will be prompted to install the required plugins. You can follow the instruction right here.
Activate License:

Right after finishing Newsy installation, please activate your theme license to receive all benefits from Newsy such as theme updates, demo import, and unlock all other features.
