Introduction

Welcome to Newsy

Newsy is completely different and extremely powerful ultimate viral theme. It’s designed in details for news, magazine, newspaper, blog and personal websites. Select between many demos, styles, templates, widgets and add-ons to create a website as you like. Publish posts and articles with Newsy easily and has access to tons of customizations and features; Beautiful design, clean coding, many exclusive plugins, lifetime updates, continuously adding features and always there support make Newsy a right choice for WordPress websites.

In Newsy documentation we try to explain all parts and options of theme and continuously add answers of frequently asked questions. Of course Newsy users may have questions not answered in this doc; feel free and start a new ticket in support center.

Get Started

  • Make sure that your Server and WordPress configuration corresponding to Newsy Requirements
  • Check out What’s Included in the Newsy Theme
  • Follow our Installing Theme Guide

Requirements for Newsy

TL;DR:
If you have installed latest version of WordPress (and it works fine), you can install Newsy too.
  • WordPress latest version
  • PHP version 7.0 or greater  (preferably use latest PHP version)
  • MySQL version 5.6 or greater (or MariaDB version 10.1 or greater)
  • PHP Configuration (recommended)
    - memory_limit 128M
    - max_execution_time 180
    - post_max_size 32M
    - upload_max_filesize 64M

What’s included

The package you download from ThemeForest.net contains following items:

  • Documentation: Offline documentation files.
  • Online Documentation: Contains address of documentation website
  • newsy.zip: Main theme. This is the file you should upload for theme installation or update
  • newsy-child.zip: This is ready-to-install child theme of Newsy.
  • Licensing: Licensing files
  • ReadMeFirst: introductions for getting started.
  • Change Log: Contains all change logs from the first version of Newsy.
Important:
Newsy plugins are bundled in main theme files. In order to access plugin files. You need to unzip newsy.zip file and open newsy/includes/plugins folder. This folder contains all plugins. Bundled and highly compatible plugins are in this folder..
