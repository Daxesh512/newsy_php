Installing Plugins

Plugins is a WordPress plugin that greatly expands the possibilities of Newsy. Newsy is bundled with several plugins such as Built In Plugins, Supported Plugin, and the most popular Recommended Plugins. All plugins are available for free including their future updates with Newsy Theme.

Note:
Please check our Plugins Overview page for more detail about all included plugins for Newsy.

How to Install Required Plugins?

Right after you finish installing & activating Newsy, you will need to install and activate Required Plugins for this theme which is a mandatory step to make Newsy fully functional. Here’s the list of all required plugins that you need to install :

  • Ak Framework (Newsy Core)
  • Newsy Elements (Content Blocks & Widgets for Newsy)

To install required plugins via TGM Plugin Activation:

  1. Please click Begin activating plugins link in prompted box or go to Appearance → Install Plugins.
  2. If you are already on the Plugin Dashboard you just need to choose all required plugins. Then hit the Install button to install and wait until installation process finish.
  3. After installation process finished, you will be prompted to return into Plugin Dashboard by hit Return to Required Plugins Installer button.
  4. Finally, you can activate the plugin by clicking Activate button.

How to Install Recommended Plugins?

Beside required plugins, you might want to install built in and recommended plugins for Newsy in order get specific functionality for your site. 

You can simply activate specific plugin that you need for your website and deactivate them if you don’t need it. You can increase your website performance by activating necessary plugins only.

There are two ways to install recommended plugins:

1. Installation via Newsy Plugin Dashboard

Install the included plugins with ease using this panel.

  1. Go to Newsy → Install Plugins page.
  2. That's all! In this page, you can install & activate your preferred plugins with one-click.

2. Installation via TGM Plugin Activation

  1. To install plugin via TGM Plugin Activation, please go to Appearance → Install Plugins 
  2. If you are already on the Plugin Dashboard you just need to choose specific plugin that you want to install. Then hit the Install button to install and wait until installation process finish.
  3. After installation process finished, you will be prompted to return into Plugin Dashboard by hit Return to Required Plugins Installer button.
  4. Finally, you can activate the plugin by clicking Activate button.
Note:
That some core (required) plugins (Ak Framework and Newsy Elements) can be locked, in that case, you need to activate different theme to unlock them.
Important:
After you activate WPBakery Page Builder plugin, you will be prompted to activate the license for this plugin.
Newsy included a license for this plugin so you don’t need to activate the license or buy the new license of this plugin. You don’t need to worry, as WPBakery Page Builder plugin will still works fine with Newsy. If there’s an update version of WPBakery Page Builder plugin, we will include it on the new version of Newsy.
