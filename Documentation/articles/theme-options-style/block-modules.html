Block & Modules

Here you can set global block settings for your site.

Blocks 

Blocks are used to build your post listings, widgets. It can be used in Page builders or sidebars.

To set global settings for your blocks follow this route:  Newsy → Theme Options →  Blocks

Block Header

In this part, you can setup several options for block header.

  1. Block Header Style
    Select a default block header style. 

  2. Block Extra Classes
    Select block extra classes. Ex: You can set boxed style block or dark style block with these predefined classes or you can add custom CSS classes.

Block Pagination

In this part, you can setup several options for block pagination.

  1. Block Loader Type
    Select a block pagination loading style.

     Load More Button

  1. Block Load More Button Width
  2. Load More Button Background Color
  3. Load More Button Text Color
  4. Load More Button Border Color
  5. Load More Button Hover Background Color
  6. Load More Button Hover Text Color
  7. Load More Button Hover Border Color 

     Next-Prev Button

  1. Next Prev Button Background Color
  2. Next Prev Button Hover Background Color
  3. Next Prev Button Border Color
  4. Next Prev Button Hover Border Color
  5. Next Prev Button Text Color
  6. Next Prev Button Hover Text Color 

Block Parts Style

In this part, you can setup several style options for block items.

  1. Category Box Style
    Override category badge styles on blocks.
  2. Module: Featured Meta Style
    Override meta wrapper style on featured image on blocks.

Block Share Buttons

In this part, you can setup several options for block social icons. Please note this option only used in Block 6 and 10.

  1. Social Icons
    Choose social icons for blocks
  2. Social Share Threshold
    Set the number of social share threshold. The total number of social share will be shown if it reaches more than this threshold.
  3. Social Icons Type
    Set which social icons style that you want to use. 
  4. Show Share Count
    Enabling this will show post share count in share box.

Modules 

Modules are represents post item for blocks. Each Module has different post elements such as post title, thumbnail, meta(author, date, views counts, share counts), categories, format icons, badges etc. You can enable disable them in this section.

To set global settings for your modules follow this route:  Newsy → Theme Options →  Modules

Module Options

  1. Title Length
  2. Show Excerpt
    • Excerpt Length
  3. Show Meta
    • Author Icon
    • Author
    • Comments
    • Shares
    • Views
    • Date
      • Date Format
      • Custom Date Format
  4. Show Categories
    • Category Type
    • Category Limit
  5. Format Icon
  6. Badge Icon
