Advertisement

Quickly place multiple ads on your site by utilizing the Google AdSense program or any other kind of ad code. 

Note:
You can place your ad in many predefined ad slots. Besides typical global locations, you can use Newsy Ad widget to have unlimited ads in your widget areas.

Available Ad Slots

To use global available ad slots for your site follow this route:  Newsy → Theme Options →  Ads.

  1. Background Ad
    Site Background Advertisement. To enable background ad you should set Full Boxed layout style and add background image in "General" tab.
  2. Header Top
    Above Header Advertisement
  3. Header Bottom
    Below Header Advertisement
  4. Footer Top
    Above Footer Advertisement
  5. Post Top
    Above Article Advertisement
  6. Archive Grid Top
    Above Archive Grid Advertisement
  7. Archive Grid Bottom
    Below Archive Grid Advertisement
  8. Post Content Top
    Above Post Content Advertisement
  9. Post Content Bottom
    Below Post Content Advertisement

Supported Ad Types

On each ad slot, there are several section option that you will find :

  1. Google AdSense Ads
    Client ID:     Insert data ads client or Google ads client content.
    Slot ID:     Insert data ads slot or Google ads slot content.
    Desktop Size:     Choose ads size for desktop device. We recommend you to use Auto on this option.
    Tablet Size:     Choose ads size for tablet device. We recommend you to use Auto on this option.
    Phone Size:     Choose ads size for mobile phone device. We recommend you to use Auto on this option.
  2. Image Ads
    Image:     Upload an image for advertisement. We recommend you to use 970×90 image size for best result.
    Image Link:     Insert url link where this advertisement image will be heading.
    Alternate Text:     Insert alternate text for advertisement image.
    Open New Tab:     Turn on this option to enable open in new tab when advertisement image clicked.
  3. Script Code
    Code: Insert Ads:Content Discovery Ads like Taboola, Outbrain or Revcontent, HTML5 Ads, Custom HTML/CSS/JS Ads and iFrames.
