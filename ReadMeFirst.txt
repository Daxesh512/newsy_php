Welcome To Newsy
 - You will need to upload newsy.zip as your theme.
 - Please install necessary plugin first. please go to https://support.akbilisim.com/docs/newsy/installing-plugins for more information.
 - Go to WordPress admin panel > Newsy > Import Demos to replicate our demo.
 - For more information, please read documentation, you can also find documentation online at : https://support.akbilisim.com/docs/newsy/introduction for more updated documentation
 - If you find any problem during installation or setting up theme, please go to https://support.akbilisim.com
 - If you going to make heavy change on themes, you can use Newsy child. But if you going to make simple css change, please use additional css feature on Customizer
